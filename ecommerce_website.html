E-Commerce Website

Developed in year 2020

Introduction

The project is about developing a responsive view E-Commerce wensite using PHP,HTML, CSS and JavaScript. This is the complete project for the web developement Advancers. With the help of this project you able to learn about various small objectives that are required in order to make any website look real and various small web applications like chap application, 2-step user verification, OTP verification and many more.


Functional Statement

    Coustomer Portal : with a responsive portal where customer register or login, can see products, categories themselves, add to their cart, can chat with the admin, can buy the product, can track his/her product.

    Admin Portal : with a responsive portal where admin can login, can change his password, can manage product , can see the profit/loss details, can change with the customer, can insert a product from his side, manage user, manage user, create category.

    Seller Portal : with a responsive navigation portal where seller can register or login, if register then can verify his phone number, can change his number can manage product which he inserted, keep an eye on delivery, can insert a product from his side, can see his profit or loss.



Requirements

    Programming Languages used : PHP, HTML, CSS, MySQL and JavaScript

    Platform used : XAMPP(control panel) for localhost connection, Notepad++
